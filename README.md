# oEmbed

**검색창에 URL을 입력하면 oEmbed API를 통해 해당 URL 웹 사이트에 포함된 콘텐츠에 대한 정보를 제공합니다.**

**우측 상단의 플로팅 배너를 통해 검색 기록을 확인할 수 있습니다.**

## 🧷 Version
- Spring Boot : 3.0.2

#

## 🧷 Dependencies
- Spring Web
- Spring Data JPA
- Spring Webflux
- H2 Database
- Json-Simple
- Thymeleaf
- Lombok
#

## 🧷 Manual

### 1. Git clone
```
git clone https://github.com/isayaksh/oEmbed.git
```

### 2. Move
```
cd oEmbed
```

### 3. Start
```
./gradlew bootRun
```

### 4. URL
```
http://localhost:8080
```
